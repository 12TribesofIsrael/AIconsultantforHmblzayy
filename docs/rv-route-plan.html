How to use this map

Tap any colored line on the map to see the recommended setup for that stretch — distance, mode, and where the RV should stage.

The default play Default to leapfrog: RV drives 10–15 mi ahead, parks at a rest area / Walmart / truck stop, Zay walks up to it, RV jumps ahead again. Switch to trail-mode only on wide-shoulder rural stretches with light traffic.

Why default leapfrog: RV creeping behind on busy roads creates close calls. Leapfrog gives you a known meet point a few miles up, and the crew can rest/cook/restock between hops instead of crawling at 3 mph all day.

Tools the route team should run

RV LIFE Pro — RV-safe GPS routing + campground/rest-area database. Plan tomorrow's stage points the night before.
AllStays Camp & RV — Walmart / Cracker Barrel / truck stop overnight + day-park database, state by state.
iExit / Trucker Path — what's at the next exit (gas, food, RV-friendly parking) so the RV can stage on short notice.

Caveat

Segments and modes are planning recommendations based on road type, terrain, and typical shoulder conditions. Confirm conditions day-of with the route team — construction, weather, or local detours can flip a trail-OK segment to leapfrog.
